# Profile overrides

The canonical house rules — section blueprint, taste budget, flattery law,
marker semantics — live in the **gh-profile** skill at
[yasyf/cc-skills](https://github.com/yasyf/cc-skills). The daily Claude
refresh installs that skill fresh on every run and follows it, so the rules
stay current without touching this repo.

This file holds only **per-profile overrides**. The refresh honors anything
listed here over the skill's defaults.
